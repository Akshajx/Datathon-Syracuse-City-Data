# GitHub Pages (`/docs`)

1. Repository **Settings → Pages**.
2. **Build and deployment**: Source = **Deploy from a branch**.
3. Branch = your working branch (e.g. `hari-local-datathon` or `main`), folder = **`/docs`**, Save.

After deployment, the site root is:

`https://<your-username>.github.io/<repository-name>/`

- **Main hub:** `index.html` — **Track B** (embedded Folium map + right sidebar to switch maps).
- **Track A:** `track-a.html` — EDA figure + notebook link.
- **Map files:** `lifescientist/*.html` (full-page if opened directly).

If the site 404s, wait 1–2 minutes and confirm the branch/folder settings.
